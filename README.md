# Opis
Bawełna Picker to edukacyjna gra typu tower‑defence, która łączy przyjemną rozgrywkę z przystępnym przedstawieniem historii walki o równouprawnienie czarnoskórych. Gracz musi pokonać nadlatujące kule bawełny — symboliczne przeszkody, które musisz za wszelką cenę zatrzymywać, rozwijając strategię i ucząc się przy tym kluczowych wydarzeń i postaci historycznych.

# Inspiracja
Pomysł na grę powstał z jednej rozmowy w klasie — dostrzegliśmy, jak często praktykowana jest w naszej klasie dyskryminacja wobec osób innego koloru skóry. Historie nierówności są pomijane, więc postanowiliśmy stworzyć grę łączącą szybką, taktyczną rozgrywkę z krótkimi lekcjami o walce o równouprawnienie.
